Block iOS Updates

iOS updates can be completely disabled with a settings profile.

The profile will disable all indicators and notifications in the Settings app that pertain to iOS updates, and will read that you are always up to date regardless of the version you have installed.

You can install this profile directly in Safari, no computer needed.

The profile can be removed at any time.
